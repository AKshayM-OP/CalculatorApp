📱 Android Dark Theme Calculator


  A simple calculator app built using Java in Android Studio with a clean dark theme interface.

------------------------------

✨ Features

  -  Dark theme UI
  -  Basic calculator functions
  -  Lightweight and beginner friendly

---------------------------------

➗ Operations

-  Addition (+)
-  Subtraction (-)
-  Multiplication (*)
-  Division (/)
-  Clear (C)

---------------------------------

🛠 Built With

-  Java
-  Android Studio
-  Android SDK
